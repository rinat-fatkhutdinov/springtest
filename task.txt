Рест сервис, работающий с базой студентов.
Необходима реализация следующих методов:
1. Заведение курса (название курса - философия, история и т.д.)
2. Получение списка курсов
3. Заведение студента (имя, фамилия)
4. Назначение курса студенту
5. Получение студентов, записанных на определенный курс
Для реализации использовать Spring boot и Spring Data. В качестве БД использовать MySQL.
Написать unit-test для каждого метода.
